# หัวข้อ พยากรณ์น้ำท่วม

=====ทำเพื่อใคร===== 
ประชากรในพื้นที่ที่เสี่ยงน้ำท่วม
เกษตกร ที่ได้รับผลกระทบ
พื้นที่ที่เกิดน้ำท่วมซ้ำและบ่อย
ประชากรที่ได้รับความเสียหายจากน้ำท่วมและต้องการความช่วยเหลือ


=====สื่งที่คาดหวัง=====
สามารถพยากรณ์น้ำท่วมได้อย่างแม่นยำ
ติดตามระดับน้ำแบบ realtime 
แจ้งเตือนประชากรในพื้นที่ได้ก่อนเหตุการณ์
สามาถส่งสัญญาณความช่วยเหลือไปยังหน่วยงานของภาครัฐได้
แสดงพื้นที่ลักษณะพื้นที่ภูมิประเทศแต่ละจังหวัดได้ เพื่อที่จะนำไปวิเคราะห์ความเสี่ยงเกิดน้ำท่วม
